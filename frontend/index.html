IEEE-BASED FEATURE EXTRACTION

30+ Features Extracted Automatically

Our system analyzes URL structure, domain properties, and content-based indicators to identify phishing attempts.

IP Address in URL

Using an IP instead of domain name is a strong phishing indicator.

@ Symbol Presence

The @ symbol in URLs forces browsers to ignore everything before it.

Hyphens in Domain

Domains with multiple hyphens often mimic legitimate sites.

HTTPS Protocol

Absence of HTTPS suggests an insecure or spoofed connection.

Special Characters

High count of special chars in URL path signals obfuscation attempts.

DNS Record

Absence of DNS records is a strong sign of a fraudulent domain.

Website Traffic

Alexa rank > 100,000 or no rank may indicate low-traffic phishing sites.

Sub-domain Depth

Multiple sub-domain levels are used to mimic legitimate URLs.

Domain Registrar

Certain low-cost/free registrars are disproportionately used by phishers.

SSL Certificate Age

Very recently issued SSL certs correlate with newly created phishing sites.

JavaScript Pop-ups

Excessive pop-up windows are used to harvest credentials deceptively.

Redirect Count

Multiple redirections obscure the final malicious destination URL.

Right-click Disabled

Disabling right-click prevents source inspection of phishing pages.

External Resources

High ratio of external assets points to a cloned website structure.

Form Action URL

Forms posting to a different domain than the page reveal data theft.

How PhishShield AI Works

classification — built on IEEE research and modern ML.

Why Random Forest
Outperforms the Rest

Random Forest is an ensemble learning method that builds multiple decision trees during training and outputs the mode of the classes for classification. This reduces overfitting significantly and provides more robust predictions across unseen phishing URLs.

  • Ensemble of 100 decision trees for each URL
  • Handles non-linear feature interactions automatically
  • Built-in feature importance ranking
  • Resistant to overfitting on training data
  • 10-fold cross-validated for reliability
